Instructions for Setting Up Forge Eclipse Environment

(this is a private page)

  1. To setup your very own forge eclipse environment, first, you have to download the forge src(as of 1.7.10 and earlier) or the MDK(as of 1.8 and later). To do this, go to files.minecraftforge.net, and click on whatever version of Minecraft you want in the top box. Then, depending on what version you want, click on the MDK or src in the "Recommended Version" box. Wait 6 seconds and press the "Skip Ad" button in the top right. BE CAREFUL NOT TO CLICK ANYTHING ON THE MAIN PAGE. IT WILL GET YOU A VIRUS! You will get a zip file. Create a new foler anywhere, and extract ALL the files in the zip into that new folder. This will be your ROOT folder.
  2. You can skip this step if you already see file extensions on the names of your files. But if you don't, and your on windows, go into file explorer and select the VIEW tab. On the right hand side, find what says "File Name Extentions." Check that off.
  3. After you've gone and done that, go into your root folder, which is the one you created in step 1, and create 3 new untitled text documents. Rename all of them to build.bat, setup workspace.bat, and setup eclipse.bat. MAKE SURE TO DELETE THE TXT OR IT WON'T WORK! If promted press yes.
  4. Right click the setup workspace batch file, and select edit. Then, type this in: gradlew setupDecompWorkspace, then save. Exit out and edit the setup eclipse batch file. Type this in: gradlew eclipse, save, and exit.
  5. Open up the setup workspace file, and wait until it exits by itself. Do the same with the setup eclipse file.
  6. After that, go to eclipse.org, and select the Download button. Download the installer, and install eclipse with the installer. When it asks for wich eclipse to download, select java development. Also make sure you have properly installed the correct version ofJDK.
  7. Make sure when you open eclipse, it says "Eclipse Mars." When it asks you for what workspace, click browse, and navigate to your root folder, and left click the "eclipse" folder ONCE. Then, press open. Then, press OK. You should be in your eclipse java workspace with all the premade tools made by forge already installed.
  8. You are now ready to start making your own mod. The eclipse classpath, or all the packages, classes and stuff in the package browser in eclipse are contained in the src folder of your root folder. If someone else gives you their src folder, and wants you to help develop, delete your current src folder and drag the new one in. It'll work the exact same way, but with all of the stuff that person already added in.
  9. This tutorial was made for windows only. If you have mac, then ask Harry.
